## 2. 진짜 원인 — 공공데이터포털 마이페이지 API공지

**[한국전력거래소] OPEN API 서비스 장애로 인한 대체서비스 안내 (2026-07-02)**
(URL: data.go.kr/bbs/ntc/selectNotice.do?originId=NOTICE_0000000004818)

> 현재 전력거래소 일부 API에 대해 서비스 장애가 발생하여 원인 확인 후 조치중.
>
> **API 장애 목록 (6건)** — 6번: **한국전력거래소_발전원별 발전량 현황조회 (미개방으로 향후 개방예정)**
>
> 5건은 대체 API(_GW) 제공. 발전원별만 대체 없이 "미개방".

**핵심:** 발전원별은 KPX가 아예 닫았고 "향후 개방예정"이라 우리가 뚫을 방법이 없다. 어제 코드만 뒤진 게 헛수고였던 이유.

**추가 발견:** 현재수급(`현재전력수급현황조회`)도 장애 목록 3번에 포함. 지금은 되지만 KPX가 대체 API로 이전 권장 → 우리도 옮기는 게 안전.

## 3. 해결책 — 대체 API 발견

data.go.kr "발전원별" 검색으로 확인한 대체 데이터셋:

### ★ 채택: 한국전력거래소_발전원별 발전량(계통기준) — 15113384
- **Base URL:** `apis.data.go.kr/B552115/PwrAmountByGen`
- **엔드포인트:** `GET /getPwrAmountByGen`
- **갱신:** 5분 단위, 실시간
- **심의:** 자동승인 (신청 즉시~20분 내 사용)
- **필드 매핑 (★기존과 다름):**
  - fuelPwr1 수력, 2 유류, 3 유연탄, 4 원자력, 5 양수, 6 가스, 7 국내탄, **8 신재생, 9 태양광**
  - **주의:** 기존 코드는 fuelPwr1~10 (8 태양광/9 풍력/10 신재생). 이 API는 **9개**이고 8·9 자리가 바뀜, **풍력 없음.** → App.jsx의 fuelNames/emissionFactors 재작성 필요.

### 참고: 현재수급 대체 API
- 한국전력거래소_현재전력수급현황조회_GW → data.go.kr/data/15158704/openapi.do
- (승인 대기 중 명세 확인 예정)

## 4. 인증키

- 공공데이터포털은 **계정당 인증키 1개를 모든 승인 API에 공용.** 기존 `.env`의 KPX_API_KEY(88자, `...uhQ==`) 그대로 대체 API에도 사용 가능.
- 도메인만 `openapi.kpx.or.kr` → `apis.data.go.kr`로 바뀜. 코드의 URL 전면 교체 필요.

## 5. 남은 작업 (순서)

1. [완료] 발전원별(계통기준) 활용신청 → **즉시 승인됨** (만료 2028-07-24).
2. [완료] `api/kpx.js` 재작성: 프론트가 path=fuel/path=sukub로 호출. fuel→apis.data.go.kr/B552115/PwrAmountByGen/getPwrAmountByGen, sukub→기존 KPX 유지. **항상 encodeURIComponent**(인코딩 버그 제거), **5분 인메모리 캐싱**(호출 한도 방어), 디버깅 console.log 삭제.
3. [완료] `App.jsx` 수정: fetch 경로 path=fuel/sukub, fuelNames·emissionFactors·COLORS를 9개 매핑으로 재작성(풍력 제거, 8 신재생·9 태양광), 디버깅 log 삭제. node --check로 api/kpx.js 문법 정상 확인.
4. [대기중] **대체 API 실호출 검증** — 승인 직후 apis.data.go.kr에서 "Forbidden"(전파 지연, 포털 공지상 최대 20분). 2026-07-24 17:32 자동 확인 스케줄(kpx-daechae-api-verify) 걸어둠. 열리면 XML 구조(item 래핑 여부) 확인 후 App.jsx 파싱 점검.
5. [미착수] 현재수급도 _GW 대체 API(15158704)로 이전할지 결정 — 지금은 기존 KPX로 동작하나 KPX가 이전 권장.
6. [미착수] cookies.txt 삭제 → git status(.env 미포함 확인) → 커밋 → Vercel 환경변수 KPX_API_KEY 등록 → 배포.

### 검증 URL (전파 확인용)
```
https://apis.data.go.kr/B552115/PwrAmountByGen/getPwrAmountByGen?serviceKey=<인코딩키>&pageNo=1&numOfRows=10&dataType=XML
```
정상 XML(fuelPwr1~9)이면 열린 것. "Forbidden"이면 아직 전파 전.
